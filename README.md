# 7.2. チュートリアル「本気でつくるチャットアプリ」
## Usage
```
$ git clone git@github.com:ionic-jp/handbook-angular-2019-chat-tutorial.git
$ cd ./handbook-angular-2019-chat-tutorial
$ npm install
$ ionic serve
```
